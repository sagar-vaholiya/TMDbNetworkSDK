# TMDbNetworkSDK

TMDbNetworkSDK is a swift framework that provides network capabilities.
It contains class which can be extended to connect and fetch data from TMDb server.
## Requirements
- iOS 10.0+
- Xcode 12.4+
- Swift 5.0+

## Installation
